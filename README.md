# rely-ui-vue

CLI para añadir componentes de **Vue 3** a tu proyecto.

## Uso

Agrega un componente al proyecto:

```bash
npx rely-ui-vue add <componente>
```

### Ejemplo

```bash
npx rely-ui-vue add button
```

Ejecuta el comando sin argumentos para ver los componentes disponibles:

```bash
npx rely-ui-vue add
```
